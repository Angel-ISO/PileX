```bash
git clone https://gitlab.com/jala-university1/cohort-3/ES.CSNT-245.GA.T2.25.M2/SB/angel.ortega/capstone.git

```

2. Install the dependencies:
```bash
npm i
```

3. Run the app:
```bash
npm start
```

and enjoy! 🚀
